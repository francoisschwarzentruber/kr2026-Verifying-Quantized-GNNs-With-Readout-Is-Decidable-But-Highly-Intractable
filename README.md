# Verifying Quantized GNNs With Readout Is Decidable But Highly Intractable

This is the supplementary material for the KR submission "Verifying Quantized GNNs With Readout Is Decidable But Highly Intractable".

## Content

- At the root, you find the pdf of the submission + appendix.
- The folder `src_check_distributivity` contains the source code for verifying distributivity over the floats or over a saturated type of int (footnote 3 p. 6, and Appendix B).
- The folder `src_verificationtool` contains the verification tool of Section 6.
- The folder `code_notebooks_csv` contains all information about the experiments which are presented in the Appendix "Experimental data and further analyses" and paper's section "Quantization Effects on Accuracy, Performance and Model Size".
